# Security and Responsible Use

This repository is an offline CUDA programming example. It does not contact remote systems, ingest credentials, or implement domain-specific targeting.

Use it only on data and workloads you own or are authorized to process. Do not add credential collection, unauthorized access, destructive behavior, or automatic interaction with third-party systems.

For a security concern in the template itself, use GitHub's private vulnerability reporting feature for this repository. For ordinary correctness issues, open a public issue with a minimal reproduction that contains no secrets or private data.
